## Prerequirements

* Visual Studio 2019
* .NET Core SDK 5.0
* SQL Server

## How To Run

* Open solution in Visual Studio 2019
* Set .Web project as Startup Project and build the project.
* Run the application.

## Set up database

* Open Package Manager Console
* Execute update-database command
